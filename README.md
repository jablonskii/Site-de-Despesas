Passos para uso do site

Usar XAMPP e copiar os arquivos para a pasta C:\xampp\htdocs\
Ligar o apache e mysql do xampp
Importar o Banco de Dados gestor_despesas.sql na conexão mysql local
Ajustar o arquivo conexao.php mudando $username = "root" e $password = "" conforme o username e senha da Conexão 
Abrir site no localhost
